# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-12-04

### Added
- Initial release of Liquid Neon Persian Picker
- Full Jalali (Persian) calendar support with `date-fns-jalali`
- Glassmorphism UI design with neon glow effects
- Dual-calendar desktop view
- Mobile bottom-sheet layout
- Month/Year quick navigation
- Date range shortcuts (Today, Yesterday, This Week, Last Month)
- Responsive design (Desktop, Tablet, Mobile, Landscape)
- Self-contained CSS file for easy integration
- TypeScript support with full type definitions
- Framer Motion animations
- RTL layout support

### Dependencies
- `date-fns-jalali`: ^4.1.0-0
- `framer-motion`: ^12.0.0
- `lucide-react`: ^0.400.0
