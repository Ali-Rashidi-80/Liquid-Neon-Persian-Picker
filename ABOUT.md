## The Philosophy
**Liquid Neon** was born from a simple observation: Most Persian (Jalali) date pickers are functional but lack soul. They feel like spreadsheets‚Äîrigid, static, and boring.

We wanted to build something different. Something that feels **alive**.

Inspired by **Cyberpunk aesthetics** and **Glassmorphism**, Liquid Neon is designed to be more than just a tool; it's a UI centerpiece. It treats light, shadow, and motion as first-class citizens, creating an interface that users actually *want* to interact with.

## Why "Liquid"?
The interface uses **fluid physics** for every interaction.
- **Spring Animations**: Menus don't just appear; they spring into place.
- **Staggered Motion**: Elements flow onto the screen sequentially.
- **Responsive Layouts**: The UI adapts fluidly from desktop to mobile, changing its entire structure (Sidebar vs. Bottom Sheet) to fit the device perfectly.

## Key Differentiators
1.  **Visual Fidelity**: We use multi-layered backdrop blurs and noise textures to create a true "frosted glass" effect that works on any background.
2.  **Neon Glow**: Interactive elements cast dynamic light, making the active state unmistakable and beautiful.
3.  **Smart UX**: We stripped away the clutter. No "Apply" buttons unless necessary. Smart inputs that format themselves. Shortcuts that actually save time.

## Tech Stack
- **React 18+**: For robust component architecture.
- **Framer Motion**: The engine behind the fluid animations.
- **Tailwind CSS**: For atomic, high-performance styling.
- **date-fns-jalali**: For bulletproof Persian calendar logic.

## For Developers, By Developers
We know how annoying it is to integrate complex UI libraries. That's why Liquid Neon is designed to be **drop-in ready**.
- **Zero Config**: Just import the component and the CSS.
- **Type Safe**: Written in TypeScript with full definitions.
- **Customizable**: CSS variables allow you to change the neon color to match your brand in seconds.
